# Travel Tracking System - User's Travel Tracker App

## Overview

The Travel Tracking System is a web application that enables users to seamlessly track and store their travel experiences using an interactive world map. The application is designed to be user-friendly and efficient, allowing users to document and share their travel journeys effortlessly.

## Features

- **Interactive World Map**: Users can mark the locations they have visited on an interactive map.
- **Travel Experience Logging**: Users can document their travel experiences, including photos, descriptions, and dates.
- **Dynamic Content Rendering**: The application uses EJS for dynamic content rendering, ensuring a responsive and engaging user interface.
- **Reliable Data Storage**: PostgreSQL is used as the database to provide reliable data storage and management capabilities.
- **Scalable Server-Side Operations**: Powered by Node.js and Express.js, the application ensures efficient and scalable server-side operations.

## Tools & Technologies

- **HTML5 & CSS3**: For creating a modern and responsive user interface.
- **Node.js**: For building the server-side logic and handling asynchronous operations.
- **Express.js**: For setting up the server and handling HTTP requests and responses.
- **EJS (Embedded JavaScript)**: For templating and dynamic content rendering.
- **PostgreSQL**: For managing and storing user data.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohdayaan786/Travel-Tracking-System.git
   cd Travel-Tracking-System
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up the database:**
   - Install PostgreSQL if you haven't already.
   - Create a new database for the application.
   - Configure the database connection settings in your environment variables.

4. **Run the application:**
   ```bash
   npm start
   ```

5. **Access the application:**
   - Open your browser and go to `http://localhost:3000`.

## Usage

- Register or log in to your account.
- Navigate to the map and start marking the locations you have visited.
- Add details about your travel experiences, including photos and descriptions.
- View and edit your travel logs anytime.
